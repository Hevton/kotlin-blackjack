## 기능 구현 목록
- [X] 카드들을 구현한다.
  - [X] 문양을 구현한다.
  - [X] 숫자를 구현한다.
- [X] 카드덱을 만든다.
- [X] 딜러랑 유저를 만든다.
  - [X] 유저 각자의 카드덱을 만든다.
- [X] 카드의 합을 구한다.
- [X] 게임 초기 세팅을 한다.
- [X] 중간결과를 반환한다.
- [X] 최종결과를 반환한다.

### 입력 구현 목록
- [X] 사람들의 이름을 입력받는다.
- [X] 카드를 더 받을지 입력 받는다.
### 출력 구현 목록
- [X] 나눠준 카드를 출력한다.
- [X] 현재 카드 상태를 출력한다.
- [X] 카드의 결과를 반환한다.
